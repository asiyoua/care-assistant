# CARE 助理 — 意图路由

根据用户意图，调用对应的子 skill。**收到触发后立即使用Skill工具调用子skill，不要自己做任何操作或回复用户。**

## 路由规则

### → care-capture
以下情况必须路由到 `care-capture`：
- 包含时间词：今天、明天、后天、下周、这周、周五、[几]号、[几]日、上午、下午、晚上、几点
- 包含动作词：开会、会议、出差、提交、完成、复习、准备、去、做、学、研究、探索、尝试
- 包含触发词：记录、待办、灵感、记一下、随手记、帮我记
- 句式为"时间+事件"或"要做某事"
- 表达了保存想法、任务、灵感的意图

### → care-review
以下情况路由到 `care-review`：
- 说"回顾一下"、"整理一下"、"周回顾"、"看看这周做了什么"
- 说"查看记录"、"有多少待办"、"查看待办"

### → care-recording
以下情况路由到 `care-recording`：
- 说"处理录音"、"录音转写"、"分析转写"、"这是今天的录音"
- 给了转写文档或.docx文件

**判断后立即使用 Skill 工具调用对应 skill，严禁自己回复用户或执行任何操作。**
